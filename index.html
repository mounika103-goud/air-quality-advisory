This repository contains an interactive Streamlit app. GitHub Pages only serves static files (HTML/CSS/JS) and cannot run Python/Streamlit apps directly.

Why you see the README:

GitHub Pages is serving the repository as a static site. To run the interactive Streamlit app you must deploy it to a platform that runs Python (for example, Streamlit Community Cloud, Render, Heroku, or Railway).

Quick options

  1. Deploy to Streamlit Community Cloud (recommended): Connect this GitHub repository at share.streamlit.io and set the main file to streamlit_app.py. Streamlit Cloud will run the app and give you a public URL.
  2. Deploy to Render / Heroku / Railway: Use the streamlit_app.py start command: streamlit run streamlit_app.py --server.port $PORT. Add a Procfile or service command accordingly.
